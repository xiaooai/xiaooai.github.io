练习1 进程创建与基本管理

  • 要求：用C创建子进程，父进程输出"Parent PID: X"，子进程输出"Child PID: Y"

  • 验收标准： 成功编译运行 父子进程信息正确 子进程先于父进程结束

解题如下：

涉及到的知识点：

  • 进程：Process ，运行中的程序

  • fork() 是linux系统调用，用于创建一个新进程，父子进程具有相同的代码和数据

  • pid==0 说明当前代码运行在子进程

  • fork 之后的代码执行了两遍，因为有两个进程

  1. 编写task1.c文件，文件内容如下：

    ​x
    #include <stdio.h> //标准输入输出
    #include <unistd.h> // 提供fork()和 getpid() 等系统调用
    #include <sys/wait.h> // 提供waitpid()函数
    

    int main() {
        pid_t pid = fork(); // 创建新进程pid，pid_t 为类型名称
        if (pid == 0) { // 在子进程中执行
            printf("Child PID: %d\n", getpid());
            _exit(0);   // 立即终止子进程
        } else if (pid > 0) {       // 父进程执行代码
            waitpid(pid, NULL, 0);  // 等待子进程
            printf("Parent PID: %d\n", getpid());
        } else {
            perror("fork");     // 打印错误信息
            return 1;
        }
        return 0;
    }

    说明如下：

    • 代码功能：创建子进程、子进程打印自己的PID后推出、父进程等待子进程结束，打印自己的PID

    • PID：进程标识符，Process IDentifier

  2. 编译运行C程序

    xxxxxxxxxx
    gcc task1.c -o task1
    ./task1

  3. 输出内容如下：

    xxxxxxxxxx
    Child PID: 142160
    Parent PID: 142159

 

练习2 内存监控工具实践

要求:

编写脚本用top/free命令监控内存使用，输出当前可用内存占比

验收标准:

  • 正确解析命令输出

  • 保留两位小数百分比

  • 支持每5秒刷新

解答如下：

  1. 创建脚本mem_monitor.sh, 内容如下：

    xxxxxxxxxx
    #!/bin/bash
    

    # 内存监控脚本
    # 功能：每5秒显示当前可用内存占总内存的百分比（保留两位小数）
    

    while true; do
        # 使用free命令获取内存信息，-m选项表示以MB为单位显示
        memory_info=$(free -m)
        
        # 提取总内存和可用内存
        total_mem=$(echo "$memory_info" | awk 'NR==2{print $2}')
        available_mem=$(echo "$memory_info" | awk 'NR==2{print $7}')
        
        # 计算可用内存百分比（保留两位小数）
        if [ "$total_mem" -ne 0 ]; then
            percent=$(echo "scale=2; $available_mem * 100 / $total_mem" | bc)
            # 格式化输出当前时间戳和百分比
            echo "$(date '+%Y-%m-%d %H:%M:%S') 可用内存占比: ${percent}%"
        else
            echo "错误：无法获取内存信息"
        fi
        
        # 等待5秒
        sleep 5
    done

    说明如下：

    • #!/bin/bash : 系统确认使用/bin/bash 来运行脚本

    • # : 表示注释，不会被执行

    • $ : 表示引用变量的值，替换值

    • memory_info=$(free -m) : free 会显示当前的内容值，-m 以MB的形式显示

    • echo "$memory_info" : 会输出变量memory_info 的内容

    • awk 'NR==2{print $2}' :

    • NR==2 : awk的内置变量NR表示当前行号

    • NR==2{print $2} : 打印第二行第二列

    • "$total_mem" -ne 0 : 检查总容量是否为零，-ne 为数值不等于比较符

    • scale=2 : 设置bc计算器的精度为2为小数‘’

    • bc : 为高精度计算器工具

  2. 添加执行权限 chmod +x mem_monitor.sh

    说明如下：

    • chmod : change Mode 修改文件权限

    • +x : execute 可执行权限

    • 默认没有权限，必须授权后才可以运行脚本

    // bytes_read: 读取的最多字节数

        ssize_t bytes_read;

            if (write(dst_fd, buffer, bytes_read) != bytes_read) {
                perror("写入目标文件失败");

                close(dst_fd);
                exit(EXIT_FAILURE);
            }
        }
    

        // 7. 异常处理
        if (bytes_read == -1) {
            perror("读取源文件失败");
            close(src_fd);
            close(dst_fd);
            exit(EXIT_FAILURE);
        }
    

        // 8. 资源清理
        close(src_fd);
        close(dst_fd);
    

        return 0;
    }

练习4：生产者—消费者问题

要求:

  • 使用信号量实现3生产者2消费者的同步模型

验收标准:

  • 缓冲区大小固定为10

  • 无数据竞争

  • 正确打印生产和消费日志

 

问题解答：

  1. 问题理解：经典的同步问题

    • 生产者 生成数据放入缓冲区

    • 消费者 从缓冲区取出数据

    • 需要保证：

      • 缓冲区满时，生产者不能继续生产

      • 缓冲区为空时，消费者不能继续消费

      • 对缓冲区的访问必须是互斥的

  2. 同步工具：

    • 互斥锁：保证对缓冲区的互斥访问，解决竞争问题，保证原子性

    • 空位信号量：跟踪缓冲区中的空槽位数量

    • 满位信号量：缓冲区中的已填充槽位数量

  3. 利用python代码来实现，内容如下：

    xxxxxxxxxx
    import threading
    import time
    import random
    

    # 缓冲区大小
    BUFFER_SIZE = 10
    # 缓冲区
    buffer = []
    

    # 工具1：互斥锁，用于保护缓冲区
    mutex = threading.Lock()
    # Semaphore 创建一个信号量
    # 工具2：空槽位信号量，此时empty的值为10
    empty = threading.Semaphore(BUFFER_SIZE)
    # 工具3：满槽位信号量，full的值为0
    full = threading.Semaphore(0)
    

    # 生产者和消费者数量
    PRODUCER_COUNT = 3
    CONSUMER_COUNT = 2
    

    # 生产项目ID
    produced_items = 0
    # 消费项目ID
    consumed_items = 0
    

    # 生产者函数
    def producer(producer_id):
        global produced_items
        while True:
            # 模拟生产时间
            # sleep：线程随机暂停一段时间
            time.sleep(random.uniform(0.1, 0.5))
            
            # 生产一个项目
            item = f"item-{produced_items}"
            produced_items += 1
            
            # 等待空槽位，成功后empty的值会减一
            empty.acquire()
            
            # 获取互斥锁
            mutex.acquire()
            
            # 将项目放入缓冲区
            buffer.append(item)
            print(f"生产者 {producer_id} 生产了 {item} (缓冲区大小: {len(buffer)})")
            
            # 释放互斥锁
            mutex.release()
            
            # 通知有新项目可用，full的值会增加1
            # release()是原子操作，线程安全
            full.release()
    

    # 消费者函数
    def consumer(consumer_id):
        global consumed_items
        while True:
            # 模拟消费时间
            time.sleep(random.uniform(0.1, 0.8))
            
            # 等待有项目可用
            full.acquire()
            
            # 获取互斥锁
            mutex.acquire()
            
            # 从缓冲区取出项目
            item = buffer.pop(0)
            consumed_items += 1
            print(f"消费者 {consumer_id} 消费了 {item} (缓冲区大小: {len(buffer)})")
            
            # 释放互斥锁
            mutex.release()
            
            # 通知有空槽位
            empty.release()
    

    # 创建并启动生产者线程
    producers = []
    for i in range(PRODUCER_COUNT):
        p = threading.Thread(target=producer, args=(i+1,))
        p.daemon = True
        producers.append(p)
        p.start()
    

    # 创建并启动消费者线程
    consumers = []
    for i in range(CONSUMER_COUNT):
        # 创建线程对象，表明线程要执行的函数是consumer
        c = threading.Thread(target=consumer, args=(i+1,))
        # 标记为守护进程，主程序退出自动退出，如监控等任务
        c.daemon = True
        # 线程对象添加进列表
        consumers.append(c)
        # 启动线程。线程是并发的，调度由操作系统觉得
        # start()：是异步的，调用后会立即返回，不会阻塞
        c.start()
    

    # 主线程等待一段时间后退出
    try:
        while True:
            time.sleep(1)
    except KeyboardInterrupt:
        print("\n程序结束")

练习5： 调度算法模拟

要求:

  • 实现FIFO和轮转调度模拟器

  • 输入进程列表，输出执行顺序

验收标准:

  • 正确处理进程到达时间

  • 可视化每个时间片状态

  • 计算平均等待时间

 

问题解答：

  1. 题目理解

    • 调度算法：多个进程需要运行，操作系统决定先运行哪个进程

    • FIFO算法：先进先出，前一个运行完，才轮到后一个

    • 轮转算法：每个乘客固定时间，时间到了就换下一个乘客

    • 关键属性：名称、何时到达、需要运行多久

  2. 编写python程序文件，内容如下：

    # name进程名称；arrival_time到达时间；burst_time执行时间；
    # remaining_time:剩余执行时间； waiting_time:等待时间；finish_time 完成时间
    class Process:
        def __init__(self, name, arrival_time, burst_time):
            self.name = name
            self.arrival_time = arrival_time
            self.burst_time = burst_time
            self.remaining_time = burst_time
            self.waiting_time = 0
            self.finish_time = 0
    

        # 使用print()函数打印时会自动触发
        def __str__(self):
            return f"{self.name} (AT: {self.arrival_time}, BT: {self.burst_time}, WT: {self.waiting_time}, FT: {self.finish_time})"
    

    # FIFO调度算法，先进先出
    def fifo_scheduler(processes):
        # 按到达时间排序
        processes.sort(key=lambda x: x.arrival_time)
        
        current_time = 0
        execution_order = []
        timeline = []
        
        for process in processes:
            # 处理空闲时间
            if current_time < process.arrival_time:
                idle_duration = process.arrival_time - current_time
                timeline.append(("Idle", idle_duration))
                current_time = process.arrival_time
            
            # 记录等待时间（考虑可能存在的负值）
            process.waiting_time = max(0, current_time - process.arrival_time)
            
            # 执行进程
            execution_order.append(process.name)
            timeline.append((process.name, process.burst_time))
            
            # 更新当前时间
            current_time += process.burst_time
            process.finish_time = current_time
        
        return execution_order, timeline, processes
    

    # 轮转调度算法，固定的时间片。没有完成，则放回到队尾。
    def round_robin_scheduler(processes, time_quantum):
        processes.sort(key=lambda x: x.arrival_time)
        queue = []
        current_time = 0
        execution_order = []
        timeline = []  # 记录时间线
        index = 0   # 已加入的进程数
        n = len(processes)
        
        # 重置进程状态
        for p in processes:
            p.remaining_time = p.burst_time
            p.waiting_time = 0
            p.finish_time = 0
        
        while True:
            # 添加所有到达的进程到队列
            # 不越界，进程已到达，添加到队列中，index加1
            while index < n and processes[index].arrival_time <= current_time:
                queue.append(processes[index])
                index += 1
            
            # 当队列为空时
            if not queue:
                # 表示还有进程没到达
                if index < n:
                    # 没有进程在队列，但有进程还未到达
                    idle_time = processes[index].arrival_time - current_time
                    timeline.append(("Idle", idle_time))
                    current_time += idle_time
                else:
                    # 所有进程都处理完毕
                    break
            else:
                # 获取队列中的第一个进程
                current_process = queue.pop(0)
                
                # 计算执行时间
                exec_time = min(time_quantum, current_process.remaining_time)
                
                # 记录执行
                execution_order.append(current_process.name)
                timeline.append((current_process.name, exec_time))
                
                # 更新剩余时间
                current_process.remaining_time -= exec_time
                current_time += exec_time
                
                # 更新其他进程的等待时间，等待时间进行更新
                for p in queue:
                    p.waiting_time += exec_time
                
                # 如果进程还未完成，放回队列
                if current_process.remaining_time > 0:
                    queue.append(current_process)
                else:
                    current_process.finish_time = current_time
        
        # 计算等待时间
        for p in processes:
            p.waiting_time = p.finish_time - p.arrival_time - p.burst_time
        
        return execution_order, timeline, processes

练习6 页面置换算法模拟

要求:

实现FIFO/LRU页面置换算法，输出缺页次数 验收标准:

  • 支持自定义页框数量(1-10)

  • 输入页面序列长度>50

  • 对比两种算法结果差异

 

解题说明

  1. 题意理解

    • 页面：虚拟内存系统中的固定大小的数据块，内存管理的基本单位。物理内存与磁盘交换数据的基本单元。

    • 虚拟内存：每个进程有独立的虚拟地址空间，页面映射将虚拟地址转化为物理地址

    • 分页：内存管理技术，将虚拟地址划分为页面，通过页表建立映射

    • 缺页异常：程序访问的页面不在物理内存中，会发生缺页异常，需要从磁盘加载到内存，满了替换页面，出现页面替换算法

    • FIFO：替换最先进入内存的页面

    • LRU：替换最近最久未使用的页面

    def fifo(frames, pages):
        page_frames = [] # 当前内存中的页面帧
        queue = []
        page_faults = 0  # 页面错误统计
        
        for page in pages:
            if page not in page_frames:
                page_faults += 1
                if len(page_frames) < frames:    # 如果还有空闲页帧
                    page_frames.append(page)    # 将页面添加到内存中
                    queue.append(page)    # 记录页面添加顺序
                else:    # 没有空闲帧
                    removed = queue.pop(0)    #移除第一个界面
                    index = page_frames.index(removed)    # 找到该界面的位置
                    page_frames[index] = page    # 替换为新页面
                    queue.append(page)    # 记录新页面的进入顺序
        return page_faults
    

    def lru(frames, pages):
        page_frames = []    # 当前页面的页面帧
        lru_list = []    # 跟踪最近使用情况
        page_faults = 0    # 页面错误计数
        
        for page in pages:
            if page in page_frames: # 如果页面在内存中
                # 更新访问顺序
                index = lru_list.index(page) # 移到队尾，表示最近访问
                lru_list.pop(index)
                lru_list.append(page)
            else: # 如果页面不在内存中
                page_faults += 1
                if len(page_frames) < frames: 
                    page_frames.append(page) 
                    lru_list.append(page) 
                else:   # 没有空闲帧
                    # 移除最近最少使用的页面
                    removed = lru_list.pop(0)
                    idx = page_frames.index(removed)
                    page_frames[idx] = page    # 替换页面
                    lru_list.append(page)
        return page_faults

    def sort_chunk(chunk_file):
        """排序单个分块文件，使用mmap加速"""
        import mmap    # 内存映射文件，用于高效文件访问
    

        with open(chunk_file, 'r+b') as f:
            # 将文件映射到内存中，向操作内存一样对文件进行读取，提高效率。虚拟地址空间技术。
            # f.fileno()是文件描述符，0表示映射整个文件，以只读方式访问映射内存

            mm.close()    # 关闭内存映射

    #define MAX_LINE 1024    // 命令行最大长度
    #define MAX_ARGS 64    // 命令参数最大数量
    

    typedef struct {
        char *args[MAX_ARGS];    // 命令参数数组
        char *input_redirect;    // 输入重定向文件名
        char *output_redirect;    // 输出重定向文件名

    // 解析命令段的函数，将输入字符串解析为命令结构体
    void parse_command_segment(char *segment, Command *cmd) {
        char *token = strtok(segment, " ");    // 利用空格分割字符串
        int arg_count = 0;    // 初始化参数计数器
        cmd->input_redirect = NULL;    // 初始化输入重定向指针为NULL
        cmd->output_redirect = NULL;    // 初始化输出重定向指针为NULL
    

        while (token != NULL && arg_count < MAX_ARGS - 1) {
            if (strcmp(token, ">") == 0) {    // 检查是否为“>”重定向符号
                token = strtok(NULL, " ");    // 继续按空格分割下一个字符串
                if (token) cmd->output_redirect = strdup(token);    //srtdup()将字符串的内容进行复制到output_redirect
            } else if (strcmp(token, "<") == 0) {
                token = strtok(NULL, " ");
                if (token) cmd->input_redirect = strdup(token);    // 同理可得
            } else {
                cmd->args[arg_count++] = strdup(token);    // 将分割得到的token加入到cmd的参数集合
            }
            token = strtok(NULL, " ");    // 获取下一个token
        }
        cmd->args[arg_count] = NULL;    // 最后将命令行参数设置为NULL
    }
    

    // 解析命令行的输入 
    // line 命令行字符串  left 左 right 右 background 指向整数的指针
    int parse_command(char *line, Command *left, Command *right, int *background) {
        *background = 0;    // 初始化
        char *last = line + strlen(line) - 1;    // 查找最后一个
        if (*last == '&') {
            *background = 1;    // 设置为后台运行
            *last = '\0';    // 替换为字符串结束符
            while (last > line && isspace(*last)) *(last--) = '\0';    // 从后遍历，移除所有字符
        }
    

        char *pipe_pos = strchr(line, '|');    // 查找管道字符 | 的位置
        if (!pipe_pos) {
            parse_command_segment(line, left);    // 如果没有找到，说明只有一个命令，存在left中
            return 1;
        } else {
            *pipe_pos = '\0';    // 将管道字符替换
            parse_command_segment(line, left);    // 管道左边的命令
            parse_command_segment(pipe_pos + 1, right);    // 管道右边的命令
            return 2;
        }
    }
    

    // 执行单个命令
    // cmd 是指向Command结构体的指针
    // background 表示是否后台运行
    void execute_single_command(Command *cmd, int background) {
        pid_t pid = fork();    // 创建子进程。返回两次，在父进程中返回子进程的ID,在子进程中返回零
        if (pid == 0) {
            if (cmd->output_redirect) {    // 检查是否有输出重定向
                int fd = open(cmd->output_redirect, O_CREAT | O_WRONLY | O_TRUNC, 0644);    // 打开输出重定向文件，设置一些权限，创建、只写、截断、0644文件权限设置
                dup2(fd, STDOUT_FILENO);    // dup2用来复制文件描述符，重定向到新打开的文件
                close(fd);    // 关闭文件描述符
            }
            if (cmd->input_redirect) {
                int fd = open(cmd->input_redirect, O_RDONLY);
                dup2(fd, STDIN_FILENO);    // 标准输入0重定向到新打开的文件
                close(fd);
            }
            execvp(cmd->args[0], cmd->args);    // 执行命令名+命令参数
            perror("exec");
            exit(EXIT_FAILURE);
        } else if (!background) {    // 0表示不后台运行，则需等待
            waitpid(pid, NULL, 0);    // NULL表示不关心子进程的退出状态
        }
    }

    void execute_pipeline(Command *left, Command *right, int background) {
        int pipefd[2];    // 定义整型数组，存储管道的文件描述符
        pipe(pipefd);    // 创建一个管道，pipefd[0]用于读取，pipefd[1]用于写入
    

        pid_t pid1 = fork();    // 创建一个子进程
        if (pid1 == 0) {
            dup2(pipefd[1], STDOUT_FILENO);    // 重定向到写入端
            close(pipefd[0]);    // 关闭读取端，不需要读取
            close(pipefd[1]);    // 关闭写入端，已完成重定向
            if (left->output_redirect) {    // 检查左命令是否有输出重定向
                int fd = open(left->output_redirect, O_CREAT | O_WRONLY | O_TRUNC, 0644);    // 打开制定的文件
                dup2(fd, STDOUT_FILENO);    // 标准的输出（如print等）重定向到打开的文件，此时输出的内容就会发到指定的文件或设备
                close(fd);
            }
            execvp(left->args[0], left->args);    // 执行左命令
            perror("exec left");
            exit(EXIT_FAILURE);
        }
    

        pid_t pid2 = fork();
        if (pid2 == 0) {
            dup2(pipefd[0], STDIN_FILENO);
            close(pipefd[0]);
            close(pipefd[1]);
            if (right->output_redirect) {
                int fd = open(right->output_redirect, O_CREAT | O_WRONLY | O_TRUNC, 0644);
                dup2(fd, STDOUT_FILENO);
                close(fd);
            }
            execvp(right->args[0], right->args);
            perror("exec right");
            exit(EXIT_FAILURE);
        }
    

        close(pipefd[0]);
        close(pipefd[1]);
    

        if (!background) {
            waitpid(pid1, NULL, 0);
            waitpid(pid2, NULL, 0);
        }
    }
    

    int main() {
        char line[MAX_LINE];   // 定义字符数组，用于存储用户输入的命令行
        Command left, right;
        int background;
    

        signal(SIGCHLD, SIG_IGN);    // 忽略子进程结束符号，系统自动回收，避免僵尸进程
    

        while (1) {

            if (!fgets(line, MAX_LINE, stdin)) break;    // 如果读取失败，跳出循环

            if (strlen(line) == 0) continue;    // 如果为空行，跳过本次循环
    

            // 解析用户输入的命令行
            int cmd_count = parse_command(line, &left, &right, &background);    
    

            if (strcmp(left.args[0], "cd") == 0) {    // 如果解析出第一个是cd
                if (left.args[1]) {    // 如果有第二个参数，跳转到要切换的目录
                    if (chdir(left.args[1]) != 0) {
                        perror("cd");
                    }
                } else { // 没有第二个参数，提示错误信息
                    fprintf(stderr, "cd: missing argument\n");
                }
            } else if (cmd_count == 1) { // 解析出只有一个命令，单命令执行
                execute_single_command(&left, background);
            } else if (cmd_count == 2) {
                execute_pipeline(&left, &right, background);
            }
        }
    

        return 0;
    }

      • 初始化inode中的原数据

      • 释放文件占用的数据块

      • 根据inode中的 数据块指针读取数据块内容

#define BLOCK_SIZE 4096    // 表示文件系统中每个数据块的大小为 4096 字节
#define MAX_BLOCKS 1024    // 表示文件系统中允许的最大数据块数量为 1024 个
#define MAX_INODES 1024    // 表示文件系统中允许的最大 inode 数量为 1024 个
#define MAX_FILESIZE (1024 * 1024)   // 表示文件系统中允许的单个文件的最大大小为 1MB
#define MAX_NAME_LEN 255    // 表示文件系统中文件名的最大长度为 255 个字符
#define DIRECT_BLOCKS 12    // 表示文件的 inode 中直接指向数据块的指针数量为 12 个
#define MAGIC_NUMBER 0xDEADBEEF    // 魔数，用于标识文件系统的类型，在挂载文件系统时可用于验证文件系统是否为预期类型


// 数据结构定义
// 定义超级块结构体，超级块包含了文件系统的全局信息
typedef struct {
    uint32_t magic;    // 魔数，用于验证文件系统的类型，确保文件系统的格式正确
    uint32_t block_size;    // 每个数据块的大小，以字节为单位
    uint32_t inode_count;    // 文件系统中inode的总数
    uint32_t free_blocks;    // 文件系统中可用的数据块数量
    uint32_t first_inode;    // 第一个inode的编号
    uint32_t first_data;    // 第一个数据块的编号
    uint32_t root_inode;    // 根目录的inode编号
    char padding[464];     // 填充字节，将结构体大小补齐到512字节，方便与磁盘扇区大小对齐
} superblock_t;


// 定义inode结构体，inode用于存储文件的元数据
typedef struct {
    uint16_t mode;    // 文件的访问权限和文件类型信息
    uint16_t uid;    // 文件所有者的用户ID
    uint32_t size;    // 文件的大小，以字节为单位
    uint32_t ctime;    // 文件创建的时间戳
    uint32_t mtime;    // 文件最后修改的时间戳
    uint32_t atime;    // 文件最后访问的时间戳
    uint32_t blocks;    // 文件占用的数据块数量
    uint32_t direct[DIRECT_BLOCKS];    // 直接指向文件数据块的数组，数组长度由DIRECT_BLOCKS宏定义
    uint32_t indirect;     // 一级间接块的编号，用于存储更多的数据块指针
    uint32_t dindirect;    // 二级间接块的编号，用于存储更多的数据块指针
    char name[MAX_NAME_LEN];    // 文件名，最大长度由MAX_NAME_LEN宏定义
} inode_t; 


// 定义目录项结构体，目录项用于表示目录中的一个条目
typedef struct {
    uint32_t inode;      // 该目录项对应的inode编号
    uint16_t rec_len;    // 该目录项的记录长度，用于在目录中定位下一个目录项
    uint8_t name_len;    // 文件名的长度
    uint8_t file_type;    // 文件的类型，如普通文件、目录、符号链接等
    char name[MAX_NAME_LEN];    // 文件名，最大长度由MAX_NAME_LEN宏定义
} dirent_t;


// 定义虚拟磁盘结构体，用于管理虚拟磁盘的状态
typedef struct {
    int fd;    // 虚拟磁盘文件的文件描述符，用于对虚拟磁盘文件进行读写操作
    superblock_t sb;    // 存储虚拟磁盘的超级块信息
    uint8_t *block_bitmap;    // 指向数据块位图的指针，用于记录哪些数据块是空闲的，哪些是已使用的
    uint8_t *inode_bitmap;    // 指向inode位图的指针，用于记录哪些inode是空闲的，哪些是已使用的
    int initialized;    // 标志位，用于表示虚拟磁盘是否已经初始化
} vdisk_t;


// 全局变量
static vdisk_t g_disk;    //该变量用于存储虚拟磁盘的相关信息，如文件描述符、超级块、位图等


// 定义一个静态函数 get_block_offset，用于计算指定数据块编号对应的磁盘偏移量
static uint32_t get_block_offset(uint32_t block_num) {
    return g_disk.sb.first_data + block_num * BLOCK_SIZE;
}


// 定义一个静态函数 get_inode_offset，用于计算指定 inode 编号对应的磁盘偏移量
static uint32_t get_inode_offset(uint32_t inode_num) {
    return g_disk.sb.first_inode + inode_num * sizeof(inode_t);
}


// 从指定编号的数据块读取数据到缓冲区
// 参数 block_num 为要读取的数据块编号，参数 buf 为用于存储读取数据的缓冲区
// 返回值：若读取成功返回 0，失败则返回 -1
static int read_block(uint32_t block_num, void *buf) {
    if (block_num >= MAX_BLOCKS) return -1;    // 检查块编号是否超出最大块数限制，若超出则返回 -1 表示错误
    off_t offset = get_block_offset(block_num);     // 调用 get_block_offset 函数计算指定块编号对应的磁盘偏移
    lseek(g_disk.fd, offset, SEEK_SET);    // 使用 lseek 函数将文件指针定位到计算得到的偏移位置
    return read(g_disk.fd, buf, BLOCK_SIZE) == BLOCK_SIZE ? 0 : -1;    // 调用 read 函数从文件描述符对应的文件中读取 BLOCK_SIZE 字节的数据到缓冲区
}


// 把缓冲区的数据写入指定编号的数据块 
// 返回值：若写入成功返回 0，失败则返回 -1
static int write_block(uint32_t block_num, const void *buf) {
    if (block_num >= MAX_BLOCKS) return -1;     // 检查块编号是否超出最大块数限制，若超出则返回 -1 表示错误
    off_t offset = get_block_offset(block_num);    // 调用 get_block_offset 函数计算指定块编号对应的磁盘偏移量
    lseek(g_disk.fd, offset, SEEK_SET);    // 使用 lseek 函数将文件指针定位到计算得到的偏移位置
    return write(g_disk.fd, buf, BLOCK_SIZE) == BLOCK_SIZE ? 0 : -1;    // 调用 write 函数将缓冲区中 BLOCK_SIZE 字节的数据写入文件描述符对应的文件
}


// 分配一个空闲的数据块
// 返回值：若成功分配则返回分配的块编号，失败则返回 -1
static int allocate_block() {
    for (uint32_t i = 0; i < MAX_BLOCKS; i++) {     // 遍历所有数据块
        if (!(g_disk.block_bitmap[i / 8] & (1 << (i % 8)))) {    // 检查位图中对应位是否为 0，若为 0 则表示该块空闲
            g_disk.block_bitmap[i / 8] |= (1 << (i % 8));    // 将位图中对应位设置为 1，表示该块已被分配
            g_disk.sb.free_blocks--;    // 减少可用块的数量
            
            // 清空新分配的块
            char zero_block[BLOCK_SIZE] = {0};    // 清空新分配的块，将一个全 0 的块写入该块编号对应的位置
            write_block(i, zero_block);
            
            // 更新超级块
            lseek(g_disk.fd, 0, SEEK_SET);    // // 更新超级块，将文件指针定位到文件开头，
            write(g_disk.fd, &g_disk.sb, sizeof(superblock_t));    // 将更新后的超级块信息写入文件
            // 返回分配的块编号
            return i;
        }
    }
    // 若未找到空闲块，返回 -1 表示分配失败
    return -1;
}


// 释放指定编号的数据块
static void free_block(uint32_t block_num) {
    if (block_num >= MAX_BLOCKS) return;    // 检查块编号是否超出最大块数限制，若超出则直接返回
    g_disk.block_bitmap[block_num / 8] &= ~(1 << (block_num % 8));    // 将位图中对应位设置为 0，表示该块已空闲
    g_disk.sb.free_blocks++;    // 增加可用块的数量
    
    // 更新超级块，将文件指针定位到文件开头，然后将更新后的超级块信息写入文件
    lseek(g_disk.fd, 0, SEEK_SET);
    write(g_disk.fd, &g_disk.sb, sizeof(superblock_t));
}


// 该函数用于分配一个空闲的inode，返回值为分配到的inode编号
static uint32_t allocate_inode() {
    for (uint32_t i = 1; i < MAX_INODES; i++) { // inode 0保留
        // [i / 8]计算在那个inode_bitmap[1]字节，i % 8 计算在字节的具体位置
        if (!(g_disk.inode_bitmap[i / 8] & (1 << (i % 8)))) {   // 检查inode位图中对应位是否为0，若为0表示该inode空闲
            g_disk.inode_bitmap[i / 8] |= (1 << (i % 8));    // 将inode位图中对应位置为1，表示该inode已被分配
            return i;
        }
    }
    return 0;
}


// 该函数用于释放指定编号的inode 
static void free_inode(uint32_t inode_num) {
    if (inode_num >= MAX_INODES) return;
    g_disk.inode_bitmap[inode_num / 8] &= ~(1 << (inode_num % 8));    //// 将inode位图中对应位置为0，表示该inode已空闲
}


// 该函数用于从磁盘中读取指定编号的inode信息到内存中
static int read_inode(uint32_t inode_num, inode_t *inode) {
    if (inode_num >= MAX_INODES) return -1;     // 检查inode编号是否超出最大inode数量限制，若超出则返回 -1 表示错误
    off_t offset = get_inode_offset(inode_num);    // 调用get_inode_offset函数计算指定inode编号对应的磁盘偏移量
    lseek(g_disk.fd, offset, SEEK_SET);    // 使用lseek函数将文件指针定位到计算得到的偏移位置
    return read(g_disk.fd, inode, sizeof(inode_t)) == sizeof(inode_t) ? 0 : -1;    // 调用read函数从文件描述符对应的文件中读取一个inode大小的数据到inode结构体中
}


// 该函数用于将内存中的inode信息写入到磁盘中指定编号的inode位置
static int write_inode(uint32_t inode_num, const inode_t *inode) {
    if (inode_num >= MAX_INODES) return -1;    // 检查inode编号是否超出最大inode数量限制，若超出则返回 -1 表示错误
    off_t offset = get_inode_offset(inode_num);    // 调用get_inode_offset函数计算指定inode编号对应的磁盘偏移量
    lseek(g_disk.fd, offset, SEEK_SET);    // 使用lseek函数将文件指针定位到计算得到的偏移位置
    return write(g_disk.fd, inode, sizeof(inode_t)) == sizeof(inode_t) ? 0 : -1;    // 调用write函数将inode结构体中的数据写入到文件描述符对应的文件中
}


// 该函数用于根据给定的路径查找对应的inode编号
// 返回值为找到的inode编号，如果查找失败则返回0
static uint32_t find_inode(const char *path) {
    if (!path || !*path || strcmp(path, "/") == 0) {    // 检查路径是否为空指针、空字符串或者是否为根路径"/"，直接返回根目录的inode编号
        return g_disk.sb.root_inode;
    }


    char *path_copy = strdup(path);    // 复制一份路径字符串，因为strtok_r会修改传入的字符串
    char *component;    // 用于存储路径中的每个组件
    char *saveptr;    // 用于strtok_r函数保存状态
    uint32_t current_inode_num = g_disk.sb.root_inode;    // 从根目录的inode编号开始查找
    
    // 逐级查找路径组件
    component = strtok_r(path_copy, "/", &saveptr);    // 使用strtok_r函数将路径按"/"分割成多个组件
    while (component) {
        inode_t current_inode;    // 用于存储当前inode的信息
        if (read_inode(current_inode_num, &current_inode) != 0) {    // 读取当前inode的信息
            fprintf(stderr, "Error: Failed to read inode %u\n", current_inode_num);
            free(path_copy);    // 释放复制的路径字符串
            return 0;
        }


        // 检查当前inode是否是目录
        if ((current_inode.mode & S_IFMT) != S_IFDIR) {
            fprintf(stderr, "Error: %s is not a directory\n", component);
            free(path_copy);    // 释放复制的路径字符串
            return 0;
        }


        // 在目录中查找下一个组件
        uint32_t next_inode_num = 0;
        char block_buf[BLOCK_SIZE];    // 用于存储从磁盘读取的目录块数据
        for (int i = 0; i < DIRECT_BLOCKS && current_inode.direct[i]; i++) {    // 遍历当前目录inode的直接数据块
            if (read_block(current_inode.direct[i], block_buf) != 0) {
                fprintf(stderr, "Error: Failed to read block %u\n", current_inode.direct[i]);    // 如果读取失败，输出错误信息并继续下一个数据块
                continue;
            }


            dirent_t *de = (dirent_t *)block_buf;    // 将数据块内容转换为目录项指针
            while ((char *)de < block_buf + BLOCK_SIZE) {    // 遍历目录项
                if (de->inode && strcmp(de->name, component) == 0) {
                    next_inode_num = de->inode;    // 找到匹配的目录项，记录下一个inode编号
                    break;
                }
                de = (dirent_t *)((char *)de + de->rec_len);    // 移动到下一个目录项
            }
            if (next_inode_num) break;// 如果找到匹配的目录项，跳出循环
        }


        if (!next_inode_num) {
            fprintf(stderr, "Error: Cannot find %s in directory\n", component);
            free(path_copy);    // 释放复制的路径字符串
            return 0;
        }


        current_inode_num = next_inode_num;    // 更新当前inode编号
        component = strtok_r(NULL, "/", &saveptr);    // 获取下一个路径组件
    }


    free(path_copy);    // 释放复制的路径字符串
    return current_inode_num;     // 返回找到的inode编号
}


// 该函数用于在指定目录的inode中添加一个新的目录项
// 目录的inode编号  新目录项的名称  新目录项对应的inode编号  新目录项的文件类型
static int add_dirent(uint32_t dir_inode_num, const char *name, uint32_t inode_num, uint8_t type) {
    inode_t dir_inode;    // 用于存储目录的inode信息
    if (read_inode(dir_inode_num, &dir_inode) != 0) return -1;    // 读取目录的inode信息
    
    // 读取目录内容
    char dir_data[BLOCK_SIZE];    // 用于存储目录的数据块内容
    if (read_block(dir_inode.direct[0], dir_data) != 0) return -1;     // 读取目录的第一个直接数据块
    
    // 查找空闲位置
    dirent_t *de = (dirent_t *)dir_data;    // 将目录数据块内容转换为目录项指针
    dirent_t *prev = NULL;    // 用于记录前一个目录项
    uint16_t needed_len = sizeof(dirent_t) + strlen(name) + 1;     // 计算新目录项所需的长度
    
    while ((char *)de < dir_data + BLOCK_SIZE) {    // 遍历目录项，查找足够大的空闲位置
        if (de->inode == 0 && de->rec_len >= needed_len) {    // 检查目录项是否空闲且长度足够
            break; // 找到空闲位置
        }
        prev = de;    // 记录前一个目录项
        de = (dirent_t *)((char *)de + de->rec_len);    // 移动到下一个目录项
    }
    
    // 如果没有足够空间，返回错误
    if ((char *)de >= dir_data + BLOCK_SIZE) return -1;
    
    // 创建新目录项
    de->inode = inode_num;    // 设置新目录项的inode编号
    de->file_type = type;    // 设置新目录项的文件类型
    de->name_len = strlen(name);    // 设置新目录项的名称长度
    strcpy(de->name, name);    // 复制新目录项的名称
    
    // 如果是合并的空闲空间，可能需要调整rec_len
    if (de->rec_len > needed_len) {
        uint16_t remaining = de->rec_len - needed_len;     // 计算剩余的空闲长度
        de->rec_len = needed_len;    // 设置当前目录项的长度为所需长度
        
        dirent_t *next = (dirent_t *)((char *)de + de->rec_len);    // 找到下一个目录项的位置
        next->inode = 0;    // 将下一个目录项标记为空闲
        next->rec_len = remaining;    // 设置下一个目录项的长度为剩余长度
        next->name_len = 0;    // 设置下一个目录项的名称长度为0 
    }
    
    // 写回目录块，将更新后的目录数据块写回磁盘
    return write_block(dir_inode.direct[0], dir_data);
}


// 该函数用于从指定目录中删除一个目录项
// 参数dir_inode_num为目录的inode编号
// 参数name为要删除的目录项名称
static int remove_dirent(uint32_t dir_inode_num, const char *name) {
    inode_t dir_inode;    // 存储目录的inode信息
    if (read_inode(dir_inode_num, &dir_inode) != 0) return -1;    //  // 读取目录的inode，检查是否存在或可读
    
    // 读取目录内容
    char dir_data[BLOCK_SIZE];
    if (read_block(dir_inode.direct[0], dir_data) != 0) return -1;
    
    // 查找目录项
    dirent_t *de = (dirent_t *)dir_data;    // 当前目录项指针
    dirent_t *prev = NULL;    // 前一个目录项指针，用于合并空闲空间
    
    while ((char *)de < dir_data + BLOCK_SIZE) {    // 遍历整个数据块
        if (de->inode != 0 && strcmp(de->name, name) == 0) {
            // 找到匹配项，标记为空闲
            de->inode = 0;
            
            // 尝试与前一个空闲项合并
            if (prev && prev->inode == 0) {
                prev->rec_len += de->rec_len;    // 合并空间：前一个目录项的长度增加当前目录项的长度
                de = prev;   // 将当前指针指向合并后的前一个目录项，避免重复处理
            }
            
            // 尝试与后一个空闲项合并
            dirent_t *next = (dirent_t *)((char *)de + de->rec_len);    // 计算下一个目录项位置
            if ((char *)next < dir_data + BLOCK_SIZE && next->inode == 0) {    
                de->rec_len += next->rec_len;    // 合并空间：当前目录项的长度增加后一个目录项的长度
            }
            
            // 写回目录块
            return write_block(dir_inode.direct[0], dir_data);
        }
        prev = de;    // 移动到下一个目录项
        de = (dirent_t *)((char *)de + de->rec_len);
    }
    // 未找到目标目录项
    return -1;
}


// 该函数用于释放inode关联的所有数据块（包括直接块和间接块）
// 参数inode为要释放块的inode结构体指针（内存中已读取的inode）
static void free_inode_blocks(inode_t *inode) {
    // 释放直接块
    for (int i = 0; i < DIRECT_BLOCKS; i++) {
        if (inode->direct[i]) {    // 若直接块指针非0（表示有效块）
            free_block(inode->direct[i]);    // 调用free_block释放该数据块
            inode->direct[i] = 0;    // 清空指针，避免悬空引用
        }
    }
    
    // 释放一级间接块
    if (inode->indirect) {    // 若存在一级间接块
        uint32_t indirect_blocks[BLOCK_SIZE / sizeof(uint32_t)];    // 定义数组存储间接块中记录的所有数据块编号
        read_block(inode->indirect, indirect_blocks);     // 读取间接块内容到数组
        
        for (int i = 0; i < BLOCK_SIZE / sizeof(uint32_t); i++) {    // 遍历间接块中记录的所有数据块
            if (indirect_blocks[i]) {    // 若块编号有效
                free_block(indirect_blocks[i]);    // 释放数据块
            }
        }
        
        free_block(inode->indirect);    // 释放间接块本身
        inode->indirect = 0;    // 清空间接块指针
    }
    
    // 释放二级间接块 (简化版未实现)
    if (inode->dindirect) {    // 若存在二级间接块（实际文件系统需递归释放）
        free_block(inode->dindirect);    // 简化处理，直接释放二级间接块本身
        inode->dindirect = 0;    // 清空指针（注：真实实现需先解析二级间接块中的一级间接块，再释放）
    }
    // 重置inode元数据：文件大小和占用块数置0
    inode->size = 0;
    inode->blocks = 0;
}


// 公共接口
// 函数 fs_init 用于初始化虚拟文件系统
// 参数 path 是虚拟文件系统对应的文件路径
// 参数 create 是一个标志，为 1 时表示创建新的文件系统，为 0 时表示读取已存在的文件系统
int fs_init(const char *path, int create) {
    
    // 以读写模式打开文件，如果 create 为真则创建文件并截断（清空）已有的内容
    g_disk.fd = open(path, O_RDWR | (create ? O_CREAT | O_TRUNC : 0), 0644);    
    if (g_disk.fd < 0) return -1;   // 如果文件打开失败，返回 -1
    
    if (create) {
        // 初始化超级块
        memset(&g_disk.sb, 0, sizeof(g_disk.sb));    // 将超级块结构体清零
        g_disk.sb.magic = MAGIC_NUMBER;    // 设置超级块的魔数，用于标识文件系统类型
        g_disk.sb.block_size = BLOCK_SIZE;    // 设置数据块大小
        g_disk.sb.inode_count = MAX_INODES;    // 设置 inode 的总数
        g_disk.sb.free_blocks = MAX_BLOCKS;    // 设置可用的数据块数量，初始化为最大块数
        g_disk.sb.first_inode = sizeof(superblock_t);    // 设置第一个 inode 的偏移量，从超级块之后开始
        g_disk.sb.first_data = g_disk.sb.first_inode + MAX_INODES * sizeof(inode_t);    // 设置第一个数据块的偏移量，在所有 inode 之后
        
        // 初始化位图
        size_t bitmap_size = (MAX_BLOCKS + 7) / 8;    // 计算块位图的大小，向上取整到字节数
        g_disk.block_bitmap = calloc(1, bitmap_size);    // 分配内存用于存储块位图，并初始化为 0
        g_disk.inode_bitmap = calloc(1, (MAX_INODES + 7) / 8);  // 分配内存用于存储 inode 位图，并初始化为 0  
        
        // 创建根目录
        g_disk.sb.root_inode = allocate_inode();    //分配一个 inode 用于根目录
        inode_t root_inode;    // 用于存储根目录的 inode 信息
        memset(&root_inode, 0, sizeof(root_inode));    // 初始化根目录的 inode 结构体，清零
        root_inode.mode = 040755; // 目录权限
        root_inode.ctime = root_inode.mtime = root_inode.atime = time(NULL);    // 设置根目录的创建时间、修改时间和访问时间为当前时间
        strcpy(root_inode.name, "/");    // 设置根目录的名称为 "/"
        
        // 为根目录分配一个数据块
        uint32_t block_num = allocate_block();
        if (block_num == (uint32_t)-1) {     // 如果分配数据块失败，关闭文件并返回 -1
            close(g_disk.fd);
            return -1;
        }
        
        root_inode.direct[0] = block_num;     // 将分配的数据块编号存储在根目录 inode 的直接块数组中
        root_inode.blocks = 1;    // 设置根目录占用的数据块数量为 1
        
        // 初始化目录内容
        char dir_data[BLOCK_SIZE] = {0};    // 创建一个大小为 BLOCK_SIZE 的字符数组，用于存储目录数据，并初始化为 0
        dirent_t *de = (dirent_t *)dir_data;    // 将缓冲区转换为目录项指针,方便操作目录项
        
        // 添加"."条目
        de->inode = g_disk.sb.root_inode;    // 设置该目录项的 inode 编号为根目录的 inode 编号
        de->rec_len = sizeof(dirent_t) + 2;    // 计算该目录项的记录长度，包括目录项结构体大小和文件名长度
        de->name_len = 1;    // 设置文件名长度为 1
        de->file_type = DT_DIR;    // 设置文件类型为目录（DT_DIR）
        strcpy(de->name, ".");    // 复制文件名 "." 到目录项的名称字段
        
        // 添加".."条目
        
        de = (dirent_t *)((char *)de + de->rec_len);    // 计算该目录项的记录长度，为数据块大小减去之前已使用的目录项的长度（即 "." 目录项的长度）
        de->inode = g_disk.sb.root_inode; // 根目录的父目录是它自己
        de->rec_len = BLOCK_SIZE - (sizeof(dirent_t) + 2);    // 计算该目录项的记录长度，为数据块大小减去之前已使用的目录项的长度（即 "." 目录项的长度
        de->name_len = 2;    // 设置文件名长度为 2
        de->file_type = DT_DIR;    // 设置文件类型为目录（DT_DIR）
        strcpy(de->name, "..");    // 复制文件名 ".." 到目录项的名称字段
        
        // 写入数据
        write_block(block_num, dir_data);    // 将目录数据写入分配的数据块
        write_inode(g_disk.sb.root_inode, &root_inode);    // 将根目录的 inode 信息写入磁盘对应的 inode 位置
        
        // 写入超级块   
        lseek(g_disk.fd, 0, SEEK_SET);    // 将文件指针定位到文件开头
        write(g_disk.fd, &g_disk.sb, sizeof(superblock_t));    // 将超级块数据写入磁盘文件的开头位置
        
        // 写入位图
        lseek(g_disk.fd, sizeof(superblock_t), SEEK_SET);    // 将文件指针定位到超级块之后的位置
        write(g_disk.fd, g_disk.inode_bitmap, (MAX_INODES + 7) / 8);    // 将 inode 位图数据写入磁盘
        write(g_disk.fd, g_disk.block_bitmap, bitmap_size);    // 将块位图数据写入磁盘
        
        g_disk.initialized = 1;    // 设置文件系统已初始化的标志位为 1
    } else {     // 如果 create 参数为 0，表示要读取现有的文件系统
        
        lseek(g_disk.fd, 0, SEEK_SET);    // 将文件指针定位到文件开头（偏移量为 0）
        if (read(g_disk.fd, &g_disk.sb, sizeof(g_disk.sb)) != sizeof(g_disk.sb)) {
            close(g_disk.fd);    // 如果读取的字节数不等于超级块的大小，说明读取失败
            return -1;
        }
        
        // 如果读取到的超级块的魔数与定义的魔数不相等，说明文件系统格式不正确
        if (g_disk.sb.magic != MAGIC_NUMBER) {
            close(g_disk.fd);   // 关闭文件描述符
            return -1;
        }
        
        // 读取位图
        size_t bitmap_size = (MAX_BLOCKS + 7) / 8;    // 计算块位图的大小
        g_disk.block_bitmap = calloc(1, bitmap_size);    // 分配内存用于存储块位图，并初始化为 0
        g_disk.inode_bitmap = calloc(1, (MAX_INODES + 7) / 8);    // 计算 inode 位图的大小（根据 inode 的数量向上取整到字节数）
        
        lseek(g_disk.fd, sizeof(superblock_t), SEEK_SET);    // 将文件指针定位到超级块之后的位置（偏移量为超级块的大小）
        read(g_disk.fd, g_disk.inode_bitmap, (MAX_INODES + 7) / 8);    // 从文件中读取 inode 位图数据到分配的内存中
        read(g_disk.fd, g_disk.block_bitmap, bitmap_size);    // 从文件中读取块位图数据到分配的内存中
        
        g_disk.initialized = 1;    // 设置文件系统已初始化的标志位为 1
    }
    
    return 0;
}


// fs_create函数用于在虚拟文件系统中创建文件或目录
int fs_create(const char *path, int is_dir) {
    if (!g_disk.initialized) return -1;    // 如果文件系统尚未初始化，返回-1
    
    // 检查路径有效性
    if (strlen(path) >= MAX_NAME_LEN) return -1;
    
    // 分配inode
    uint32_t inum = allocate_inode();
    if (inum == 0) return -1;     // 如果分配inode失败（返回0表示失败），返回-1
    
    // 初始化inode
    inode_t ino;    // 用于存储新创建的inode信息
    memset(&ino, 0, sizeof(ino));    // 初始化inode结构体，将其内容清零
    ino.mode = is_dir ? 040755 : 0100644;    // 根据is_dir标志设置inode的权限，目录权限为040755，文件权限为0100644
    ino.ctime = ino.mtime = ino.atime = time(NULL);     // 设置inode的创建时间、修改时间和访问时间为当前时间
    
    char *name = strrchr(path, '/');    // 从路径中提取文件名（找到最后一个斜杠的位置）
    if (!name) name = (char *)path;     // 如果路径中没有斜杠，那么整个路径就是文件名
    else name++;// 如果有斜杠，文件名是斜杠后面的部分
    strncpy(ino.name, name, MAX_NAME_LEN);   // 将文件名复制到inode的name字段中，最多复制MAX_NAME_LEN个字符 
    
    // 如果是目录，分配一个块并初始化
    if (is_dir) {
        uint32_t block_num = allocate_block();    // 为目录分配一个数据块
        if (block_num == (uint32_t)-1) { 
            free_inode(inum);     // 如果分配数据块失败（返回(uint32_t)-1表示失败），释放已分配的inode并返回-1
            return -1;
        }
        
        ino.direct[0] = block_num;    // 将分配的数据块编号存储在inode的直接块数组的第一个位置
        ino.blocks = 1;    // 设置目录占用的数据块数量为1
        
        // 初始化目录内容
        char dir_data[BLOCK_SIZE] = {0};    // 初始化目录内容的缓冲区
        dirent_t *de = (dirent_t *)dir_data;    // 将缓冲区转换为目录项指针
        
        // 添加"."条目
        de->inode = inum;    // 设置 inode 号为当前目录的 inode
        de->rec_len = sizeof(dirent_t) + 2;    // 记录长度 = 目录项大小 + 2（对齐填充）
        de->name_len = 1;     // 名称长度 = 1（`.`）
        de->file_type = DT_DIR;    // 文件类型 = 目录
        strcpy(de->name, ".");    // 名称 = "."
        
        // 添加".."条目
        de = (dirent_t *)((char *)de + de->rec_len);    // 移动到下一个目录项位置
        de->inode = g_disk.sb.root_inode; // 简化版，所有子目录的父目录都是根目录
        de->rec_len = BLOCK_SIZE - (sizeof(dirent_t) + 2);    // 剩余空间全部给 `..`
        de->name_len = 2;    // 名称长度 = 2（`..`）
        de->file_type = DT_DIR;    // 文件类型 = 目录
        strcpy(de->name, "..");    // 名称 = ".."
        
        write_block(block_num, dir_data);    // 将目录内容写入分配的数据块
    }
    
    // 写入inode
    write_inode(inum, &ino);
    
    // 添加到父目录
    char parent_path[MAX_NAME_LEN] = {0};     // 用于存储父目录的路径
    char *last_slash = strrchr(path, '/');     // 找到路径中最后一个斜杠的位置
    if (last_slash) {    // 如果路径中有斜杠
        strncpy(parent_path, path, last_slash - path);    // 复制路径中斜杠前面的部分作为父目录路径
        if (strlen(parent_path) == 0) strcpy(parent_path, "/");     // 如果复制后的父目录路径长度为0，说明是根目录，设置为 "/"
    } else {
        strcpy(parent_path, "/");    // 如果路径中没有斜杠，说明文件或目录在根目录下，设置父目录路径为 "/"
    }
    
    // 根据父目录路径查找父目录的inode编号
    uint32_t parent_inum = find_inode(parent_path);
    if (parent_inum == 0) {    // 如果查找父目录的inode失败（返回0表示失败），释放已分配的inode和数据块
        free_inode(inum);
        if (is_dir) free_block(ino.direct[0]);
        return -1;
    }
    // 将新创建的文件或目录添加到父目录中
    if (add_dirent(parent_inum, name, inum, is_dir ? DT_DIR : DT_REG) != 0) {
        free_inode(inum);     // 如果添加失败，释放已分配的inode和数据块（如果是目录）并返回-1
        if (is_dir) free_block(ino.direct[0]);
        return -1;
    }
    
    return 0;
}


// 从指定路径的文件中读取数据到缓冲区
// 参数 size 是要读取的字节数, 参数 offset 是从文件开始的偏移量
int fs_read(const char *path, char *buf, size_t size, off_t offset) {
    if (!g_disk.initialized) return -1;    // 检查文件系统是否已初始化，若未初始化则返回 -1
    
    uint32_t inum = find_inode(path);    // 根据路径查找文件的 inode 编号
    if (inum == 0) return -1;
    
    inode_t ino;    // 用于存储文件的 inode 信息
    if (read_inode(inum, &ino) != 0) return -1;    // 读取文件的 inode 信息，若读取失败则返回 -1
    
    if (offset >= ino.size) return 0;    // 若偏移量大于等于文件大小，说明没有数据可读取，返回 0
    if (offset + size > ino.size) size = ino.size - offset;    // 若要读取的字节数加上偏移量超过文件大小，调整要读取的字节数
    
    size_t remaining = size;     // 剩余要读取的字节数
    size_t copied = 0;    // 已经复制到缓冲区的字节数
    uint32_t block_offset = offset / BLOCK_SIZE;    // 数据所在的块偏移量
    uint32_t intra_block_offset = offset % BLOCK_SIZE;    // 数据在块内的偏移量
    
    // 读取直接块
    while (block_offset < DIRECT_BLOCKS && remaining > 0 && ino.direct[block_offset]) {
        char block_data[BLOCK_SIZE];   // 用于存储从块中读取的数据
        if (read_block(ino.direct[block_offset], block_data) != 0) break;
        
        size_t to_copy = BLOCK_SIZE - intra_block_offset;    // 计算本次要复制的字节数
        if (to_copy > remaining) to_copy = remaining;
        
        memcpy(buf + copied, block_data + intra_block_offset, to_copy);    // 将块数据复制到缓冲区
        copied += to_copy;    // 更新已复制的字节数
        remaining -= to_copy;    // 更新剩余要读取的字节数
        block_offset++;    // 移动到下一个块
        intra_block_offset = 0;    // 重置块内偏移量
    }
    
    // 读取间接块 (简化版未实现)
    
    // 更新访问时间
    ino.atime = time(NULL);    // 更新文件的访问时间为当前时间
    write_inode(inum, &ino);    // 将更新后的 inode 信息写回磁盘
    
    return copied;    // 返回实际读取的字节数
}


// 向指定路径的文件中写入数据
int fs_write(const char *path, const char *buf, size_t size, off_t offset) {
    if (!g_disk.initialized) return -1;
    
    uint32_t inum = find_inode(path);
    if (inum == 0) return -1;
    
    inode_t ino;
    if (read_inode(inum, &ino) != 0) return -1;
    
    // 检查文件大小限制
    if (offset + size > MAX_FILESIZE) {
        size = MAX_FILESIZE - offset;
        if (size <= 0) return -1;    // 若调整后要写入的字节数小于等于 0，则返回 -1
    }
    
    size_t remaining = size;    // 剩余要写入的字节数
    size_t written = 0;     // 已经写入的字节数
    uint32_t block_offset = offset / BLOCK_SIZE;     // 数据所在的块偏移量
    uint32_t intra_block_offset = offset % BLOCK_SIZE;    // 数据在块内的偏移量
    
    // 写入直接块
    while (block_offset < DIRECT_BLOCKS && remaining > 0) {
        if (!ino.direct[block_offset]) {    // 若当前块未分配，则分配一个新的块
            uint32_t new_block = allocate_block();
            if (new_block == (uint32_t)-1) break;    // 若分配块失败，则跳出循环
            ino.direct[block_offset] = new_block;    // 将新分配的块编号存储到 inode 的直接块数组中
            ino.blocks++;    // 增加文件占用的块数
        }
        
        char block_data[BLOCK_SIZE] = {0};    // 用于存储要写入块的数据
        if (intra_block_offset != 0 || remaining < BLOCK_SIZE) {
            read_block(ino.direct[block_offset], block_data); // 若块内偏移量不为 0 或者剩余要写入的字节数小于块大小，读取块的现有数据
        }
        
        size_t to_write = BLOCK_SIZE - intra_block_offset;    // 计算本次要写入的字节数
        if (to_write > remaining) to_write = remaining;
        
        // 将数据从缓冲区复制到块数据中
        memcpy(block_data + intra_block_offset, buf + written, to_write);
        // 将更新后的块数据写回磁盘，若写入失败则跳出循环
        if (write_block(ino.direct[block_offset], block_data) != 0) break;
        
        written += to_write;    // 更新已写入的字节数
        remaining -= to_write;    // 更新剩余要写入的字节数
        block_offset++;    // 移动到下一个块
        intra_block_offset = 0;    // 重置块内偏移量s
    }
    
    // 更新inode信息,若写入后文件大小增加，更新文件大小
    if (offset + written > ino.size) {
        ino.size = offset + written;
    }
    ino.mtime = time(NULL);    // 更新文件的修改时间为当前时间
    write_inode(inum, &ino);    // 将更新后的 inode 信息写回磁盘
    
    return written;    // 返回实际写入的字节数
}


// 函数用于删除指定路径的文件或目录
int fs_unlink(const char *path) {
    if (!g_disk.initialized) return -1;
    
    uint32_t inum = find_inode(path);
    if (inum == 0) return -1;
    
    // 获取文件名和父目录路径
    char *name = strrchr(path, '/');     // 查找路径中最后一个斜杠的位置
    if (!name) name = (char *)path;
    else name++;    // 如果有斜杠，文件名是斜杠后面的部分
    
    char parent_path[MAX_NAME_LEN] = {0};    // 用于存储父目录路径
    char *last_slash = strrchr(path, '/');    // 再次查找路径中最后一个斜杠的位置
    if (last_slash) {    // 如果路径中有斜杠
        strncpy(parent_path, path, last_slash - path);    // 复制斜杠前面的部分作为父目录路径
        if (strlen(parent_path) == 0) strcpy(parent_path, "/");
    } else {
        strcpy(parent_path, "/");
    }
    
    uint32_t parent_inum = find_inode(parent_path);    // 根据父目录路径查找父目录的 inode 编号
    if (parent_inum == 0) return -1;
    
    // 读取inode
    inode_t ino;
    if (read_inode(inum, &ino) != 0) return -1;
    
    // 如果是目录，检查是否为空 (简化版只检查基本项)
    if ((ino.mode & 040000) && ino.direct[0]) {    // 通过 inode 的 mode 判断是否为目录（S_IFDIR 掩码），并且检查是否有数据块
        char dir_data[BLOCK_SIZE];
        read_block(ino.direct[0], dir_data);    // 读取目录的数据块内容
        
        dirent_t *de = (dirent_t *)dir_data;    // 将目录数据块内容转换为目录项指针
        int count = 0;      // 用于计数非空目录项的数量
        while ((char *)de < dir_data + BLOCK_SIZE) {
            if (de->inode != 0) count++;   // 如果目录项的 inode 编号不为 0，说明该目录项非空
            de = (dirent_t *)((char *)de + de->rec_len);    // 移动到下一个目录项
        }
        
        if (count > 2) { // 有除了.和..之外的条目
            return -1;    // 说明目录不为空
        }
    }
    
    // 释放数据块
    free_inode_blocks(&ino);
    
    // 从父目录中删除
    if (remove_dirent(parent_inum, name) != 0) return -1;
    
    // 释放inode
    free_inode(inum);
    memset(&ino, 0, sizeof(ino));
    write_inode(inum, &ino);
    
    return 0;
}


// 函数用于关闭文件系统，释放相关资源
void fs_shutdown() {
    if (g_disk.initialized) {
        free(g_disk.block_bitmap);    // 释放块位图占用的内存
        free(g_disk.inode_bitmap);    // 释放 inode 位图占用的内存
        close(g_disk.fd);    // 关闭文件描述符
        memset(&g_disk, 0, sizeof(g_disk));    // 将全局的虚拟磁盘结构体清零，释放相关资源
    }
}


// 在原有代码基础上增加以下函数
// 函数功能：列出指定路径下目录的内容
// 参数 path：要列出内容的目录路径
void fs_ls(const char *path) {
    // 检查文件系统是否已初始化，如果未初始化，打印提示信息并返回
    if (!g_disk.initialized) {
        printf("Filesystem not initialized\n");
        return;
    }


    // 根据路径查找对应的 inode 编号
    uint32_t inum = find_inode(path);
    // 如果未找到对应的 inode 编号，打印提示信息并返回
    if (inum == 0) {
        printf("Directory not found: %s\n", path);
        return;
    }


    // 用于存储查找到的 inode 信息
    inode_t ino;
    // 读取 inode 信息，如果读取失败，打印提示信息并返回
    if (read_inode(inum, &ino) != 0) {
        printf("Failed to read inode\n");
        return;
    }


    // 检查该 inode 对应的是否为目录，通过 inode 的 mode 与 040000 按位与判断
    if (!(ino.mode & 040000)) {
        printf("Not a directory: %s\n", path);
        return;
    }


    // 打印目录内容的提示信息和表头
    printf("Contents of %s:\n", path);
    printf("Mode\tInode\tSize\tName\n");


    // 用于存储目录数据块的内容
    char dir_data[BLOCK_SIZE];
    // 读取目录的第一个直接数据块，如果读取失败，打印提示信息并返回
    if (read_block(ino.direct[0], dir_data) != 0) {
        printf("Failed to read directory block\n");
        return;
    }


    // 将目录数据块内容转换为目录项指针
    dirent_t *de = (dirent_t *)dir_data;
    // 遍历目录项
    while ((char *)de < dir_data + BLOCK_SIZE) {
        // 如果目录项的 inode 编号不为 0，表示该目录项有效
        if (de->inode != 0) {
            // 用于存储子目录项或文件项的 inode 信息
            inode_t child_ino;
            // 读取子目录项或文件项的 inode 信息
            if (read_inode(de->inode, &child_ino) == 0) {
                // 打印子目录项或文件项的权限、inode 编号、大小和名称
                printf("%06o\t%d\t%u\t%s\n", 
                       child_ino.mode, de->inode, child_ino.size, de->name);
            }
        }
        // 移动到下一个目录项
        de = (dirent_t *)((char *)de + de->rec_len);
    }
}


// 函数功能：获取并打印指定路径文件或目录的详细信息
// 参数 path：要获取信息的文件或目录路径
void fs_stat(const char *path) {
    // 检查文件系统是否已初始化，如果未初始化，打印提示信息并返回
    if (!g_disk.initialized) {
        printf("Filesystem not initialized\n");
        return;
    }


    // 根据路径查找对应的 inode 编号
    uint32_t inum = find_inode(path);
    // 如果未找到对应的 inode 编号，打印提示信息并返回
    if (inum == 0) {
        printf("File not found: %s\n", path);
        return;
    }


    // 用于存储查找到的 inode 信息
    inode_t ino;
    // 读取 inode 信息，如果读取失败，打印提示信息并返回
    if (read_inode(inum, &ino) != 0) {
        printf("Failed to read inode\n");
        return;
    }


    // 打印文件或目录的统计信息提示
    printf("Statistics for %s:\n", path);
    // 打印 inode 编号
    printf("Inode: %u\n", inum);
    // 打印权限（以八进制显示）
    printf("Mode: %06o\n", ino.mode);
    // 打印文件或目录的大小
    printf("Size: %u bytes\n", ino.size);
    // 打印占用的数据块数量
    printf("Blocks: %u\n", ino.blocks);
    // 打印创建时间，使用 ctime 函数将时间戳转换为字符串格式
    printf("Created: %s", ctime((time_t *)&ino.ctime));
    // 打印修改时间
    printf("Modified: %s", ctime((time_t *)&ino.mtime));
    // 打印访问时间
    printf("Accessed: %s", ctime((time_t *)&ino.atime));


    // 打印直接数据块的提示
    printf("Direct blocks: ");
    // 遍历 inode 中记录的直接数据块编号数组
    for (int i = 0; i < DIRECT_BLOCKS; i++) {
        // 如果直接数据块编号有效，打印该编号
        if (ino.direct[i]) printf("%u ", ino.direct[i]);
    }
    // 换行
    printf("\n");


    // 如果存在一级间接数据块
    if (ino.indirect) {
        // 打印一级间接数据块的编号
        printf("Indirect block: %u\n", ino.indirect);
        // 用于存储一级间接数据块中记录的其他数据块编号
        uint32_t indirect_blocks[BLOCK_SIZE / sizeof(uint32_t)];
        // 读取一级间接数据块的内容到数组中
        read_block(ino.indirect, indirect_blocks);


        // 打印一级间接数据块中记录的其他数据块编号的提示
        printf("Indirect blocks: ");
        // 遍历数组，打印有效的数据块编号
        for (int i = 0; i < BLOCK_SIZE / sizeof(uint32_t); i++) {
            if (indirect_blocks[i]) printf("%u ", indirect_blocks[i]);
        }
        // 换行
        printf("\n");
    }
}
